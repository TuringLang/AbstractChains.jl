AbstractChains.AbstractChain — Type
AbstractChains.AbstractChain

An abstract type for Markov chains, i.e., a data structure which stores samples obtained from Markov chain Monte Carlo (MCMC) sampling.

Markov chains should generally have dictionary-like behaviour, where keys are mapped to matrices of values.

Note that, despite its name, an AbstractChain can store multiple chains (e.g. when sampling in parallel, or when combining chains from multiple runs). The main reason why this needs to be a singular form is because the package name is already AbstractChains (plural).

Interface

To implement a new subtype of AbstractChain, you need to define the following methods:

  • Base.size should return a tuple of ints (the exact meaning is left to you)
  • Base.keys should return a list of keys
  • AbstractChains.get_data(chn, key)
  • AbstractChains.iter_indices(chn)
  • AbstractChains.chain_indices(chn)

You can optionally define the following methods for efficiency:

  • AbstractChains.niters(chn)
  • AbstractChains.nchains(chn)

AbstractChains.chain_indices — Function
AbstractChains.chain_indices(chn)

Obtain the indices of each chain in the AbstractChain object chn.

This function should return an AbstractVector{<:Integer}.

source
AbstractChains.get_data — Function
AbstractChains.get_data(chn, key)

Obtain the data associated with key from the AbstractChain object chn.

This function should return an AbstractMatrix where the rows correspond to iterations and columns correspond to chains.

source
AbstractChains.iter_indices — Function
AbstractChains.iter_indices(chn)

Obtain the indices of each iteration for the AbstractChain object chn.

This function should return an AbstractVector{<:Integer}.

AbstractChains.nchains — Method
AbstractChains.nchains(chn)

Obtain the number of chains in the AbstractChain object chn.

The default implementation calculates the length of AbstractChains.chain_indices(chn). You can define your own method if you have a more efficient way of obtaining this information.

AbstractChains.niters — Method
AbstractChains.niters(chn)

Obtain the number of iterations in the AbstractChain object chn.

The default implementation calculates the length of AbstractChains.iter_indices(chn). You can define your own method if you have a more efficient way of obtaining this information.
